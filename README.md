# PwC Call Centre Trends Dashboard

Tool:Power BI  
Data:PwC Switzerland Virtual Internship Dataset (Forage)  
Status:Completed  

## Business Problem
Built an operational dashboard to monitor call centre KPIs 
for PwC's client engagement team — tracking agent performance, 
call resolution rates, customer satisfaction scores, and 
monthly volume trends.

## Dashboard KPIs
| Metric | Value |
|--------|-------|
| Overall Customer Satisfaction | 3.4 / 5 |
| Total Calls Answered | 4,054 |
| Total Calls Abandoned | 946 |
| Average Speed of Answer | 67.52 seconds |
| Average Talk Duration | 3 min 45 sec |
| Overall Resolution Rate | 72.9% |
